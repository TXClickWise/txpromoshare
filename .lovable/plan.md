## Phase 1: Foundation & Core UI

Given the enormous scope, this first build will establish:

### 1. Design System
- Premium hospitality-friendly aesthetic with warm tones
- Dutch language throughout
- Custom tokens, typography, components

### 2. Pages Built
- **Landing page** - conversion-focused, Dutch
- **Pricing page** - Free/Basic/Pro plans
- **Login & Register** - auth-ready shells
- **Dashboard** - main workspace overview
- **Event overview** - list/grid of events
- **Create/Edit event** - full event form with all fields
- **Distribution center** - sharing hub
- **Settings** - workspace/branding placeholder

### 3. Architecture Prepared
- Multi-tenant data model concepts (tenant context)
- Role-based navigation guards
- Subscription plan gating structure
- Extensible routing with layout system
- i18n-ready string constants (Dutch default)

### 4. NOT in Phase 1 (future iterations)
- Backend/database (Lovable Cloud)
- Actual auth flow
- Widget embed system
- ClickWise/HighLevel integration
- Media library
- Team management
- Super admin backend
- Ticketing architecture
- Calendar integration

This gives you a polished, commercially-feeling frontend to iterate on. Backend and real data can be added next.
